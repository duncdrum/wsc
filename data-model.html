The WSC Database: Introduction

The orignal WSC Database was a relational database with three main entity tables, supplemented by minor datatables.
All records in the databases are interrelated. The three main tables contained:

  • Bibliographical data
  • Biographical data
  • Terminological data

The current version of the database, adheres to the basic entity classes, and can be qeuried online. The data, however, has been remodeled and edited to be more easily accesible and both human and machine readable.
